# A股波段交易筛选系统 v4.0.0

## 🎯 系统定位

专业的A股波段交易筛选系统，专注于**主板+创业板融资融券标的**，采用严格风控策略，适合稳健型投资者。

---

## ✨ 核心策略

### 📋 筛选标准

| 维度 | 要求 | 说明 |
|------|------|------|
| **板块** | 主板 + 创业板 | 排除科创板（风险过高） |
| **融资融券** | 必须 | 机构参与度高，流动性好 |
| **ST/亏损** | 排除 | 严格风控，避免地雷股 |
| **市值** | ≤160亿 | 中小市值，成长空间大 |
| **涨幅** | -2% ~ 5% | 不追涨，寻找波段买点 |
| **量比** | 1.5 ~ 3.0 | 温和放量，避免异常 |
| **持仓** | 最多3只 | 分散风险，集中优势 |

### 🎲 波段交易逻辑

```
买入时机：
  ✅ 回调到位（-2% ~ 0%）：最佳波段买点
  ✅ 温和上涨（0% ~ 3%）：趋势确认
  ✅ 涨幅适中（3% ~ 5%）：可考虑

避免追涨：
  ❌ 涨幅>7%：追高风险大
  ❌ 量比>5：异常放量，警惕出货
  ❌ 换手>15%：可能主力出货
```

## 📊 评分体系

### 多维度智能评分（满分100分）

1. **融资融券（40%权重）**
   - 融资融券评分 ≥70分：+28分
   - 融资净流入 >0.05亿：+15分
   - 融资净流出 <-0.05亿：-10分

2. **涨跌幅（波段偏好）**
   - 回调到位（-2% ~ 0%）：+20分 ⭐最佳
   - 温和上涨（0% ~ 3%）：+15分
   - 涨幅适中（3% ~ 5%）：+5分
   - 涨幅过大（>7%）：-20分

3. **量比健康度**
   - 1.5 ~ 2.5：+15分（最佳）
   - 2.5 ~ 3.5：+8分
   - >5：-10分（异常）

4. **市值（成长性）**
   - 50 ~ 100亿：+15分（最佳）
   - 100 ~ 160亿：+10分
   - >160亿：-20分（超限）

5. **资金流向**
   - 主力强力流入：+20分
   - 主力温和流入：+10分
   - 主力强力流出：-20分

6. **换手率**
   - 3% ~ 8%：+10分（适中）
   - >15%：-15分（过高）

7. **板块加分**
   - 创业板：+5分（成长性）

**入选标准：评分 ≥50分**

## 🚀 使用指南

### 启动服务

```bash
# 后端服务（端口8000）
python backend/main.py

# 前端服务（端口5173）
cd frontend
npm run dev
```

### API接口

#### 1. 波段交易筛选（推荐）
```
GET /api/band-trading
```

**参数：**
- `change_min`: 涨幅下限（默认-2%）
- `change_max`: 涨幅上限（默认5%）
- `volume_ratio_min`: 量比下限（默认1.5）
- `volume_ratio_max`: 量比上限（默认3.0）
- `market_cap_max`: 市值上限（默认160亿）
- `limit`: 返回数量（最多3只）

**示例：**
```bash
curl "http://localhost:8000/api/band-trading?limit=3"
```

#### 2. 通用筛选（兼容）
```
GET /api/screen
```
自动调用波段交易筛选，参数同上。

#### 3. 实时行情
```
GET /api/realtime?code=000001
```

## ⚠️ 风险控制

### 严格排除

- ❌ 科创板（688xxx）
- ❌ ST股票（*ST、ST、S开头）
- ❌ 亏损股票
- ❌ 非融资融券标的
- ❌ 市值>160亿
- ❌ 涨幅>7%（追高风险）

### 持仓管理

- 📊 **最多3只个股**：分散风险
- 🎯 **波段操作**：不追涨，寻找回调买点
- 💰 **融资融券**：机构参与，流动性保障
- 📈 **中小市值**：成长空间大

## 📝 使用建议

### 1. 筛选时机
- ✅ 大盘回调时：寻找超跌反弹机会
- ✅ 个股回调时：波段买点
- ❌ 大盘暴涨时：避免追高

### 2. 买入策略
- 优先选择评分≥70分的股票
- 关注融资净流入的标的
- 回调到位（-2%~0%）时买入最佳

### 3. 卖出策略
- 涨幅达到5-7%考虑减仓
- 出现融资净流出信号及时止盈
- 跌破买入价3%止损

### 4. 持仓管理
- 每次最多持有3只
- 单只仓位不超过总资金的33%
- 保持一定现金仓位应对机会
