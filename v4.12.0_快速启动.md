# v4.12.0 快速启动指南

## 🚀 5分钟快速上手

### 1. 启动系统（2分钟）

```bash
# 终端1: 启动后端
cd backend
python main.py

# 终端2: 启动前端
cd frontend
npm run dev
```

访问：http://localhost:5174

---

### 2. 设置第一个提醒（1分钟）

1. 点击顶部 **🔔 提醒中心** 按钮
2. 点击 **+ 添加提醒**
3. 填写表单：
   - 股票代码：000001
   - 股票名称：平安银行
   - 目标价格：13.00
   - 方向：上涨到
4. 点击 **创建提醒**

✅ 完成！当股价达到13元时，你会收到通知。

---

### 3. 添加第一个持仓（2分钟）

1. 点击顶部 **📊 我的持仓** 按钮
2. 点击 **+ 添加持仓**
3. 填写表单：
   - 股票代码：000001
   - 股票名称：平安银行
   - 买入价格：12.00
   - 数量：1000
   - 买入日期：今天
4. 勾选 **自动计算止损止盈**
5. 点击 **添加持仓**

✅ 完成！系统会自动追踪盈亏，并在触及止损/止盈时提醒你。

---

## 🎯 核心功能

### 提醒中心 🔔
- **价格提醒**: 股价达到目标价时通知
- **止损止盈**: 自动监控持仓盈亏
- **异动提醒**: 涨跌幅或成交量异常时通知
- **提醒历史**: 查看所有触发的提醒

### 持仓追踪 📊
- **持仓管理**: 添加、删除、查看持仓
- **盈亏计算**: 实时显示盈亏金额和比例
- **统计分析**: 总市值、总盈亏、持仓数量
- **风险评估**: 集中度、板块分散度、优化建议

---

## 💡 使用技巧

### 1. 自动止损止盈
添加持仓时勾选"自动计算止损止盈"，系统会：
- 自动设置止损价（买入价 × 95%）
- 自动设置止盈价（买入价 × 115%）
- 自动创建价格提醒

### 2. 查看风险评估
在持仓列表中查看"优化建议"面板，了解：
- 持仓集中度
- 板块分散度
- 行业分散度
- 个性化优化建议

### 3. 快捷键
- `Ctrl+Enter`: 开始筛选
- `Ctrl+F`: 精选过滤
- `Ctrl+R`: 重置
- `Esc`: 取消分析

## ⚠️ 注意事项

### 浏览器通知权限
首次使用时，浏览器会请求通知权限，请点击"允许"。

### 数据存储
所有数据保存在浏览器 localStorage，清除浏览器数据会导致数据丢失。

### 价格更新
当前版本使用模拟价格更新（±2%波动），实际部署需连接真实API。
