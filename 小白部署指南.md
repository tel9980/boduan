# 📖 小白部署指南 - 零基础也能用

> 本指南专为完全不懂技术的小白用户设计，只需要跟着步骤操作即可！

## 🎯 你将得到什么？

一个**自动筛选股票**的系统，帮你找到适合波段交易的股票：
- ✅ 自动筛选4900+只股票
- ✅ AI智能分析推荐理由
- ✅ 提供买入价、止损价、目标价
- ✅ 三种策略可选（激进/保守/平衡）

## 📋 准备工作（5分钟）

### 第一步：下载代码

1. 打开网址：https://github.com/tel9980/boduan
2. 点击绿色按钮 **"Code"**
3. 点击 **"Download ZIP"**
4. 下载完成后，解压到任意文件夹（比如：`D:\股票筛选系统`）

### 第二步：安装 Python

1. 打开网址：https://www.python.org/downloads/
2. 点击黄色按钮 **"Download Python"**
3. 下载完成后，双击安装
4. **重要**：安装时勾选 **"Add Python to PATH"**（在最下面）
5. 点击 **"Install Now"**

### 第三步：安装 Node.js

1. 打开网址：https://nodejs.org/
2. 点击左边的绿色按钮（推荐版本）
3. 下载完成后，双击安装
4. 一路点击 **"下一步"** 即可

### 第四步：获取 AI 密钥（可选，但强烈推荐）

1. 打开网址：https://open.bigmodel.cn/
2. 注册账号并登录
3. 点击右上角头像 → **"API密钥"**
4. 点击 **"创建新的API密钥"**
5. 复制密钥（类似：`abc123def456...`）
6. 打开项目文件夹中的 `.env` 文件
7. 将密钥粘贴到 `GLM_API_KEY=` 后面

## 🚀 一键安装（10分钟）

### 方式1：使用一键安装脚本（推荐）

1. 打开项目文件夹
2. 双击运行 **`一键安装.bat`**
3. 等待安装完成（约5-10分钟）
4. 看到 **"安装完成！"** 即可

### 方式2：手动安装

如果一键安装失败，可以手动安装：

#### 安装后端依赖
1. 按住 `Shift` 键，在项目文件夹空白处右键
2. 点击 **"在此处打开 PowerShell 窗口"**
3. 输入以下命令并回车：
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r backend\requirements.txt
```

#### 安装前端依赖
1. 在 PowerShell 中继续输入：
```bash
cd frontend
npm install
```

## ❓ 常见问题

### Q1: 安装时提示"找不到 Python"？
**A**: 重新安装 Python，记得勾选 **"Add Python to PATH"**

### Q2: 安装时提示"找不到 npm"？
**A**: 重新安装 Node.js，安装完成后重启电脑

### Q3: 启动后浏览器打不开？
**A**: 
1. 检查后端窗口是否显示 **"Uvicorn running on http://0.0.0.0:8000"**
2. 检查前端窗口是否显示 **"Local: http://localhost:5173/"**
3. 如果都正常，手动在浏览器输入：http://localhost:5173

### Q4: 点击筛选按钮没反应？
**A**: 
1. 打开浏览器控制台（按 F12）
2. 查看是否有错误信息
3. 检查后端窗口是否有错误信息

### Q5: 筛选速度太慢？
**A**: 
1. 使用 **"启动系统_极速版.bat"** 启动
2. 首次需要等待10-12分钟预热
3. 预热完成后，任何时候点击都能秒开

### Q6: 没有 AI 分析？
**A**: 
1. 检查 `.env` 文件中是否配置了 `GLM_API_KEY`
2. 检查密钥是否正确
3. 如果不需要 AI 分析，可以忽略

### Q7: 如何停止系统？
**A**: 
1. 关闭所有命令行窗口
2. 或者在窗口中按 `Ctrl + C`

### Q8: 如何更新系统？
**A**: 
1. 下载最新代码
2. 解压覆盖原文件夹
3. 重新运行 **"一键安装.bat"**
4. 重新启动系统

## 🎓 进阶使用

### 自定义筛选条件

如果你想修改筛选条件（需要一点技术基础）：

1. 打开 `backend/main.py` 文件
2. 找到 `BAND_TRADING_CONFIG` 配置
3. 修改参数：
   - `max_market_cap`: 市值上限（默认160亿）
   - `change_range`: 涨跌幅范围（默认-2%到5%）
   - `volume_ratio_range`: 量比范围（默认1.5到3.0）

### 调整缓存时间

如果你想调整缓存刷新间隔：

1. 打开 `backend/auto_preheat.py` 文件
2. 找到 `schedule.every(30).minutes`
3. 修改数字（单位：分钟）
