# 📊 A股股票智能筛选器 (波段交易增强版)

基于量价分析与技术指标的A股精选系统，专为波段交易者设计，帮助投资者快速筛选出符合特定条件的潜力股票。

## ✨ 核心功能

### 🚀 波段交易模式 (New!)
专为稳健投资者设计的**融资融券标的**筛选模式：
- **严选标的**：只做A股主板和创业板的**融资融券**标的，严格排除ST、*ST及亏损股。
- **市值控制**：流通市值 **≤160亿**，专注中小盘活跃股。
- **拒绝追高**：涨幅限制在 **-2% 至 +5%**，寻找回调低吸或启动初期的机会。
- **技术共振**：集成 **RSI(6日)**、**MACD**、**KDJ** 多维度指标验证，确保趋势向上。
- **小白友好**：
  - **智能评分**：0-100分直观评价，80分以上重点关注。
  - **中文标签**：自动打标“资金强势”、“趋势向上”、“超卖反弹”、“主板/创业板”。
  - **卡片视图**：清晰展示核心数据与AI简评，告别枯燥表格。

### 🔍 经典筛选模式
- **涨幅范围**：2% - 6%（适中涨幅）
- **量比范围**：1.5 - 3（资金关注）
- **市值范围**：80 - 350亿（中盘股）
- **精选逻辑**：阶梯式放量 + 站稳5日线 + 热门板块

## 🛠️ 技术栈

### 后端
- **FastAPI** - 高性能 Python Web 框架
- **新浪财经 API** - 实时股票行情数据源
- **AkShare** - 历史行情与融资融券数据
- **Pandas / NumPy** - 专业金融数据分析

### 前端
- **React 19** - 用户界面框架
- **TypeScript** - 类型安全的 JavaScript
- **Vite** - 快速的前端构建工具
- **Axios** - HTTP 客户端

## 🚀 快速开始

### 环境要求
- Python 3.10+
- Node.js 18+

### 启动服务
1. **后端**:
   ```bash
   cd backend
   pip install -r requirements.txt
   python -m uvicorn main:app --reload
   ```

2. **前端**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. **访问**: 打开浏览器访问 `http://localhost:5173`
