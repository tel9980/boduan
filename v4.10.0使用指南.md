# v4.10.0 使用指南

## 🎯 六大新功能快速上手

### 1️⃣ 数据新鲜度提醒（v4.9.0）

**位置**：筛选结果统计面板右上角

**如何使用**：
1. 查看颜色标识：
   - 🟢 绿色 = 数据新鲜（<5分钟）→ 可放心使用
   - 🟡 黄色 = 数据稍旧（5-15分钟）→ 建议关注
   - 🔴 红色 = 数据较旧（>15分钟）→ 立即刷新

2. 当显示红色时，点击"🔄 立即刷新"按钮获取最新数据

**小贴士**：
- 开盘时段（9:30-15:00）建议每5-10分钟刷新一次
- 收盘后数据更新频率降低，可适当延长刷新间隔

---

### 2️⃣ 智能持仓建议（v4.9.0）

**位置**：板块/行业分布下方的粉色卡片

**如何使用**：
1. 查看每只股票的建议持仓比例（如：35%、40%、25%）
2. 查看组合总风险评级：
   - ✅ 适合买入 = 可按建议比例操作
   - ⚠️ 谨慎操作 = 降低仓位或等待
   - ❌ 建议观望 = 暂不操作

3. 阅读底部的具体建议

**实战示例**：
```
假设你有10万元：
- 宏柏新材 35% = 3.5万元
- 鑫铂股份 40% = 4万元
- 华民股份 25% = 2.5万元
```

**小贴士**：
- 严格按比例分配，不要随意调整
- 每只股票设置-5%止损位
- 达到目标价位及时止盈

---

### 3️⃣ 风险分散评分（v4.9.0）

**位置**：板块/行业分布右侧的紫色卡片

**如何使用**：
1. 查看综合评分（0-100分）：
   - 80-100分（优秀）= 风险充分分散 ✅
   - 60-79分（良好）= 风险较好分散 👍
   - 40-59分（一般）= 需要优化 ⚠️
   - 0-39分（较差）= 建议重新筛选 ❌

2. 查看各维度评分：
   - 板块分散度：是否覆盖沪市、深市、创业板
   - 行业分散度：是否选择不同行业
   - 市值分散度：市值是否差异合理
   - 涨幅分散度：涨幅是否差异适中

3. 阅读底部的优化建议

**评分解读**：
- 板块分散度 100分 = 3个板块各1只 ✅
- 行业分散度 100分 = 3个不同行业 ✅
- 市值分散度 >60分 = 市值差异合理 ✅
- 涨幅分散度 >60分 = 涨幅差异适中 ✅

**小贴士**：
- 综合评分<60分时，建议重新筛选
- 优先关注板块和行业分散度（权重各35%）
- 避免选择同一行业的多只股票

---

### 4️⃣ 板块轮动提示（v4.10.0）

**位置**：板块分布卡片顶部

**如何使用**：
1. 查看板块轮动提示：
   - 📈 资金流入的板块（+X只）
   - 📉 资金流出的板块（-X只）

2. 阅读智能建议：
   - 系统会提示关注强势板块

**实战示例**：
```
🔄 板块轮动提示
📈 创业板 资金流入 [+1只]
📉 沪市主板 资金流出 [-1只]
💡 建议：关注创业板的强势股票
```

**小贴士**：
- 板块轮动是市场资金流向的重要信号
- 资金流入的板块往往有更好的上涨机会
- 结合行业热度一起分析效果更佳

---

### 5️⃣ 行业热度排行（v4.10.0）

**位置**：行业分布卡片顶部

**如何使用**：
1. 查看Top 3行业排行：
   - 🥇 金牌：第1名（热度最高）
   - 🥈 银牌：第2名
   - 🥉 铜牌：第3名

2. 查看每个行业的详细数据：
   - 平均涨幅
   - 平均量比

3. 阅读智能建议：
   - 系统会提示优先关注热度最高的行业

**实战示例**：
```
🔥 行业热度 Top 3
🥇 1 化工    涨幅 4.8%  量比 1.9
🥈 2 有色    涨幅 4.7%  量比 1.8
🥉 3 综合    涨幅 4.7%  量比 1.6
💡 建议：优先关注 化工 行业
```

**小贴士**：
- 热度评分 = 平均涨幅 × 60% + (平均量比 - 1) × 20
- 热度最高的行业往往有更好的赚钱效应
- 结合板块轮动一起分析效果更佳

---

### 6️⃣ 快速对比模式（v4.10.0）

**位置**：初步筛选结果区域右上角

**如何使用**：
1. 当股票数量在2-5只时，会显示"⚖️ 快速对比"按钮
2. 点击按钮，展开并排对比视图
3. 查看7个关键指标的对比：
   - 评分（综合评分）
   - 涨幅（涨跌幅）
   - 量比（成交量比）
   - 市值（流通市值）
   - 板块（所属板块）
   - 行业（所属行业）
   - 风险（风险等级）

4. 金色边框标识各项最优值：
   - 👑 评分最高
   - 🔥 涨幅最高
   - 📊 量比最高
   - 💎 市值最小

5. 阅读底部的对比结论

**实战示例**：
```
⚖️ 快速对比模式
┌─────┬──────────┬──────────┬──────────┐
│指标 │宏柏新材  │鑫铂股份  │华民股份  │
├─────┼──────────┼──────────┼──────────┤
│评分 │75分      │80分 👑   │65分      │
│涨幅 │+4.75%    │+4.78% 🔥 │+4.77%    │
│量比 │1.94 📊   │1.79      │1.60      │
└─────┴──────────┴──────────┴──────────┘
💡 对比结论：鑫铂股份 综合评分最高（80分），建议优先关注。
```

**小贴士**：
- 快速对比模式仅在2-5只股票时显示
- 金色边框帮助你快速识别最优指标
- 综合评分最高的股票通常是最佳选择

## 📋 完整操作流程

### 步骤1：开始筛选
1. 点击"开始筛选"按钮
2. 等待8-12秒获取数据

### 步骤2：查看数据新鲜度
1. 查看右上角的颜色标识
2. 如果是红色，点击"立即刷新"

### 步骤3：分析板块轮动
1. 查看板块轮动提示
2. 识别资金流入/流出的板块
3. 关注强势板块

### 步骤4：分析行业热度
1. 查看行业热度Top 3
2. 识别热度最高的行业
3. 优先关注热门行业

### 步骤5：评估风险分散
1. 查看风险分散评分
2. 如果<60分，考虑重新筛选
3. 如果≥60分，继续下一步

### 步骤6：快速对比股票
1. 点击"快速对比"按钮
2. 查看并排对比视图
3. 识别金色边框标识的最优指标
4. 阅读对比结论

### 步骤7：确定持仓比例
1. 查看智能持仓建议
2. 记录每只股票的建议比例
3. 查看组合总风险

### 步骤8：执行操作
1. 如果组合风险为"低"，按建议比例买入
2. 如果组合风险为"中"，降低仓位或等待
3. 如果组合风险为"高"，暂不操作

## ⚠️ 风险提示

1. **数据时效性**：
   - 只使用绿色或黄色标识的数据
   - 红色数据必须刷新后再用

2. **板块轮动**：
   - 板块轮动是参考信号，不是绝对指标
   - 结合行业热度和个股质量综合判断

3. **行业热度**：
   - 热度高不代表一定能赚钱
   - 注意行业的持续性和基本面

4. **快速对比**：
   - 金色边框只标识单项最优
   - 综合评分最高才是最佳选择

5. **持仓比例**：
   - 严格按建议比例分配
   - 不要全仓单只股票
   - 预留10-20%现金应急

6. **风险控制**：
   - 每只股票设置-5%止损
   - 达到目标价位及时止盈
   - 不要频繁交易

7. **市场环境**：
   - 关注大盘走势
   - 避免在大跌时买入
   - 保持理性，不追涨杀跌

## 💡 常见问题

**Q1：数据一直是红色怎么办？**
A：点击"立即刷新"按钮，或检查后台筛选任务是否正常运行。

**Q2：板块轮动提示不显示怎么办？**
A：需要至少筛选2次才会显示轮动提示，第一次筛选时没有历史数据对比。

**Q3：行业热度排行显示的是什么？**
A：显示当前筛选结果中热度最高的3个行业，热度 = 平均涨幅 × 60% + (平均量比 - 1) × 20。

**Q4：快速对比按钮不显示怎么办？**
A：快速对比仅在股票数量2-5只时显示，超过5只或少于2只都不会显示。

**Q5：持仓比例可以自己调整吗？**
A：不建议。系统基于评分自动计算，随意调整可能增加风险。

**Q6：风险分散评分多少算合格？**
A：≥60分为合格，≥80分为优秀。<60分建议重新筛选。

**Q7：组合风险为"高"还能买吗？**
A：不建议。高风险意味着市场波动大或股票质量参差不齐，建议观望。

**Q8：如何提高风险分散评分？**
A：
- 确保选择不同板块的股票
- 确保选择不同行业的股票
- 避免市值过于集中
- 避免涨幅过于集中

## 🎓 进阶技巧

### 1. 最佳操作时间
- 开盘后30分钟（9:30-10:00）
- 尾盘前30分钟（14:30-15:00）

### 2. 板块轮动策略
- 关注连续2-3次流入的板块
- 避免追逐单次流入的板块
- 结合大盘走势判断轮动持续性

### 3. 行业热度策略
- 优先选择Top 1行业的股票
- 避免选择热度排名靠后的行业
- 关注行业热度的变化趋势

### 4. 快速对比技巧
- 优先关注综合评分最高的股票
- 金色边框越多，单项优势越明显
- 结合风险等级综合判断

### 5. 仓位管理
- 初始仓位：50-60%
- 盈利后：可加仓至70-80%
- 亏损后：降仓至30-40%

### 6. 止盈止损
- 止损：-5%（严格执行）
- 止盈：+5%~+8%（根据量比判断）
- 移动止损：盈利>3%后，止损位上移至成本价

### 7. 组合调整
- 每周评估一次持仓
- 及时替换表现不佳的股票
- 保持风险分散评分≥60分

### 8. 综合分析
- 数据新鲜度 + 板块轮动 + 行业热度 = 市场方向
- 风险分散评分 + 快速对比 = 选股质量
- 智能持仓建议 = 资金分配方案
