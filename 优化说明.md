# 波段交易系统优化说明 v4.1.0

## 🎯 核心优化

### 1. 更智能的融资融券评分
- **覆盖率提升**：从66%提升到70%的股票支持融资融券
- **评分更科学**：
  - 融资余额权重30%
  - 净流入权重40%（最关键）
  - 融资占比权重30%
- **基础分提高**：从50分提升到55分

### 2. 更精准的波段买点识别
**涨跌幅评分优化**：
- 📉📉 深度回调（-2%到-0.5%）：+25分 → **黄金买点**
- 📉 小幅回调（-0.5%到0%）：+20分 → **优质买点**
- 📈 温和上涨（0%到2%）：+18分 → 趋势良好
- ⚡ 适度上涨（2%到4%）：+10分
- ⚠️ 涨幅过大（>7%）：-25分 → **严重减分，避免追高**

### 3. 更严格的筛选标准
- **评分门槛**：从50分提升到55分
- **风险等级**：
  - 🏆 低风险：≥70分
  - 💎 中风险：55-69分
  - ⚠️ 高风险：<55分

### 4. 更直观的视觉展示
**股票卡片优化**：
- 动态边框颜色（根据评分）
- 悬停动画效果
- 融资融券信息突出显示
- 推荐理由和风险提示分类展示

## 📊 评分权重分配

| 维度 | 权重 | 说明 |

| 涨跌幅 | 25% | 偏好回调，不追涨 |
| 量比 | 15% | 1.5-2.2为完美量比 |
| 市值 | 10% | 偏好40-120亿 |
| 资金流向 | 10% | 主力流入加分 |
| 换手率 | 5% | 2-6%为完美换手 |

## 🎨 视觉优化

### 评分等级标识
- **≥80分**：🏆 优质（绿色边框）
- **≥70分**：💎 良好（蓝色边框）
- **≥60分**：⚡ 一般（橙色边框）
- **<60分**：⚠️ 较差（红色边框）

### 融资融券展示
- 蓝色背景突出显示
- 显示余额、净流、占比三个关键指标
- 评分直接展示

## 🚀 使用建议

### 波段交易策略
1. **优先选择回调股票**：-2%到0%的回调是最佳买点
2. **必须融资融券标的**：确保流动性和市场认可度
3. **市值适中**：40-120亿市值成长空间大
4. **量比健康**：1.5-2.5倍量比最佳
5. **主力流入**：资金流向为正更安全

### 风险控制
- 每次最多持仓3只
- 市值不超过160亿
- 排除ST和亏损股
- 避免追涨（涨幅>5%减分）
- 评分<55分不建议买入
